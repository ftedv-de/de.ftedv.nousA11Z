# Changelog

## 1.0.0

- Initial release for NOUS A11Z `TS011F` / `_TZ3210_6cmeijtd`.
- Added three independent outlet controls.
- Added Tuya-style Basic cluster initialization including manufacturer attribute `0xfffe`.
- Added power, voltage, current and energy readings from endpoint 1.
- Added optional debug logging.
- Added Homey Flow triggers, actions and conditions for all three outlets.
- Added app and device SVG icons.
